# Web Components

- design pattern for reusable custom elements using HTML and js
- shadow DOM

## Pros VS React

- native
- framework agnostic
- use directly with other DOM APIs

## Cons VS React

- complex, low-level standard API
- third-party ecosystem still growing
- not many UI kits
- much less popular
